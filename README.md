# 📝 To-Do List Web App

A simple and modern **To-Do List Web Application** built using **HTML, CSS, and JavaScript**.  
This project helps users manage their daily tasks by adding, completing, updating and deleting tasks easily.

---

## 🚀 Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 💾 Tasks saved using **LocalStorage**
- 🎨 Simple and responsive UI

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)
- LocalStorage API
